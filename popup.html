Chrome Extension

Markdown Review Overlay

Inline review actions for Markdown rich diff on GitHub and GitHub Enterprise.

Privacy

  • No external server calls
  • No analytics, tracking, or telemetry
  • Posts only to the currently opened GitHub or GHE host

Scope

  • Works on PR Files changed pages
  • Targets Markdown rich diff blocks
  • Uses your existing authenticated browser session
  • Shows host-local HUD and per-file readiness badges

Inspired by the ideas in rich-diff-comments. See bundled notices for MIT attribution.
